Matter over Wi-Fi Demo Overview

This document walks through the steps to build the Matter 1.1 and 1.2 applications, such as Lighting, Light Switch, Window Covering, Thermostat, Door Lock and On/Off Plug for both the NCP and SoC examples on the EFR32MG24 and SiWx917 SoC boards. A complete list of hardware supported for Wi-Fi is included on the Hardware Requirements page.

Before running the Matter Wi-Fi demo or developing for Wi-Fi make sure that you have all the required hardware and software for each use case.
